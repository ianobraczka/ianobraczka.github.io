# Ian Obraczka — Portfolio

Personal portfolio site — a single place for **projects**, **background**, **contact**, and links to **GitHub** and **LinkedIn**. Static and straightforward: more landing page than documentation.

## Tech Stack

**HTML**, **CSS**, and a bit of **JavaScript** (no framework in this repo). **Inter** via Google Fonts. Multilingual copy on the pages (**PT / EN / ES**) with simple client-side switching.

## What’s here

- **Projects** (`index.html`) — featured write-ups for **Meetal**, **Vandal**, and **ReUsa**. **EasyPlay** and **uniflix** stay in the markup as archive-style sections (hidden in the normal scroll experience). Images load from **`assets/images/`**.
- **About** (`about.html`), **Contact** (`contact.html`), and **Résumé** (`cv.html`) — the usual portfolio pages; résumé PDFs and exports live under **`assets/documents/`** (the CV page uses print-to-PDF from the HTML).

## Hosting

Published with **GitHub Pages** from this **`ianobraczka.github.io`** repository — typically at **https://ianobraczka.github.io**. No other deploy config lives in the tree.
